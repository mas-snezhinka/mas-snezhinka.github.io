«Снежинка» — это проект МФТИ, реализуемый при поддержке Министерства науки и высшего образования РФ, Министерства РФ по развитию Дальнего Востока и Арктики, а также Правительства Ямало-Ненецкого автономного округа (ЯНАО).
На международном уровне проект изначально продвигался через Арктический совет, а сейчас активно развивается в рамках научно-технологического сотрудничества стран БРИКС.
